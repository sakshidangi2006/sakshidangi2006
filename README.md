## Featured Projects

### 🤖 AI Resume Analyzer
> Upload your resume, set your target role — get AI-powered feedback instantly.

A full-stack web app that parses **PDF and DOCX** resumes, sends them to an AI API with a target job role and job description, and returns structured analysis with improvement suggestions. Includes user authentication, a personal history dashboard, and per-report delete functionality.

**Tech:** `Python` `Flask` `SQLAlchemy` `PyPDF2` `python-docx` `AI API` `HTML` `CSS` `JavaScript`

🔗 [View Repository](https://github.com/sakshidangi2006/AI_Resume_Analyzer)

---

### 🪞 Mood Mirror
> An AI-powered mood journal that reflects your emotional patterns back to you.

A full-stack Flask app where users log journal-style mood entries and receive AI-generated insights and reflections. Features user auth, SQLite-backed mood history, and trend tracking over time.

**Tech:** `Python` `Flask` `SQLite` `AI API (Anthropic/OpenAI)` `HTML` `CSS` `JavaScript`

🔗 [View Repository](https://github.com/sakshidangi2006/Mood_Mirror)
